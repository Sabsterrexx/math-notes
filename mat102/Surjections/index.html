How to Prove a Function is Surjective

Those of you who are taking MAT102 currently or in the future may be struggling with how to prove a function is surjective. I've made this guide in an effort to hopefully reduce that anxiety and help you actually understand the concept of a surjective function.

Definition of Surjectivity

The definition of surjectivity is:

\[ \text{If we have a function } f: A \to B, \text{ then } B \subseteq f(A). \]

By the definition of what it means to be a subset, this means that every element of B is an element of f(A).

Using quantifiers, this can be written as:

\[ \forall z \, (z \in B \implies z \in f(A)). \]

More rigorously, surjectivity means:

\[ \forall y \in B, \, \exists x \in A \text{ such that } f(x) = y. \]

General Proof Structure

The most common way to prove surjectivity is to use the following proof structure:

  1. Let \(y \in B\).
  2. Let \(x =\) "an expression in terms of \(y\)" such that \(x \in A\) (so that \(x\) is a valid input for \(f\)). In most cases, a good choice is setting \(x\) to the "inverse" of the function.
  3. Show that \(f(x) = y\) when plugged in.

Example of a Surjectivity Proof

Consider the function:

\[ \text{Let } f: \mathbb{R} \to [0, \infty), \quad f(x) = x^2. \]

We will prove that \(f\) is surjective.

Proof

Let

\[ y \in [0, \infty). \]

Since

\[ y \geq 0, \]

we know that

\[ \sqrt{y} \in \mathbb{R} \]

(no undefined or imaginary value in the domain/input of \(f\)), so let

\[ x = \sqrt{y}. \]

Now compute:

\[ f(x) = (\sqrt{y})^2 = y. \]

Thus, for all \(y \in [0, \infty)\), there exists \(x \in \mathbb{R}\) such that

\[ f(x) = y. \]

Therefore, \(f\) is surjective. QED.

How to Disprove Surjectivity

To disprove surjectivity, it is enough to show that there exists an element of B that is not in the image of \(f\).

In quantifiers, this is the negation of the surjectivity statement:

\[ \neg \left( \forall y \in B, \, \exists x \in A \text{ such that } f(x) = y \right). \]

Which is equivalent to:

\[ \exists y \in B \text{ such that } \forall x \in A, \, f(x) \neq y. \]

If you can prove this, then \(f\) is not surjective.

Intuition

Intuitively, disproving surjectivity means:

Pick a value in the codomain that the function never outputs.

Example of a Non-Surjective Function

Consider the function:

\[ \text{Let } f: \mathbb{Z} \to \mathbb{Z}, \quad f(x) = 2x. \]

We will prove that \(f\) is not surjective.

Proof

Let

\[ y = 1 \]

so that

\[ y \in \mathbb{Z}. \]

For any \(x \in \mathbb{Z}\),

\[ f(x) = 2x \]

is always even, so there is no \(x\) such that

\[ f(x) = 1 = y, \]

since \(1\) is odd.

Thus, there exists \(y \in \mathbb{Z}\) such that for all \(x \in \mathbb{Z}\),

\[ f(x) \neq y. \]

Therefore, \(f\) is not surjective. QED.

Important Note About the Image of a Function

You may realize that

\[ f(A) = \{ f(x) \mid x \in A \} \]

is just the image (range) of the function by definition.

So another way to define surjectivity is to say:

\[ f(A) = B. \]

Since

\[ B \subseteq f(A) \]

and

\[ f(A) \subseteq B, \]

but because the image is always a subset of the codomain of a function,

\[ f(A) \subseteq B \]

is trivial.

So we really only need to prove:

\[ B \subseteq f(A). \]

Final Thoughts

This is just a general approach, and not every question will be solvable by this exact method, but it works for most cases.

Regardless, a single algorithm does not exist that works for all surjectivity proofs. Sometimes you will have to use your head more and work with the existing definitions you have.

As a wise man once said:

"Definition. Definition is always your friend."
