Those of you who are taking MAT102 currently or in the future may be struggling with how to prove a function is surjective. I've made this guide in an effort to hopefully reduce that anxiety and help you actually understand the concept of a surjective function.

\bigskip

The definition of surjectivity is:

\text{If we have a function } f: A \to B, \text{ then } B \subseteq f(A).

By the definition of what it means to be a subset, this means that every element of B is an element of f(A).

Using quantifiers, this can be written as:

\forall z \, (z \in B \implies z \in f(A)).

More rigorously, surjectivity means:

\forall y \in B, \, \exists x \in A \text{ such that } f(x) = y.

\bigskip

The most common way to prove this is to use the following proof structure:

  1. Let y \in B.
  2. Let x = an expression in terms of y such that x \in A (so that x is a valid input for f), and show that f(x) = y when plugged in. In most cases, a good choice is setting x to the "inverse" of the function.

\bigskip

These steps may be confusing to understand, so here's a simple example:

\text{Let } f: \mathbb{R} \to [0, \infty), \quad f(x) = x^2. \text{ Prove that } f \text{ is surjective.}

Proof:

Let y \in [0, \infty).

Since y \geq 0, we know that \sqrt{y} \in \mathbb{R} (no undefined/imaginary value in the domain/input of f), so let x = \sqrt{y}.

f(x) = (\sqrt{y})^2 = y.

Thus, for all y \in [0, \infty), there exists x \in \mathbb{R} such that f(x) = y.

Therefore, f is surjective. QED.

\bigskip

To disprove surjectivity, it is enough to show that there exists an element of B that is not in the image of f. In quantifiers, this is the negation of the above statement:

\neg \left( \forall y \in B, \, \exists x \in A \text{ such that } f(x) = y \right).

Which is equivalent to:

\exists y \in B \text{ such that } \forall x \in A, \, f(x) \neq y.

If you can prove this, then f is not surjective.

\bigskip

Intuitively, this means: pick a value in the codomain that the function never outputs.

\bigskip

To make things clearer, here's a simple example:

\text{Let } f: \mathbb{Z} \to \mathbb{Z}, \quad f(x) = 2x. \text{ Prove that } f \text{ is not surjective.}

Proof:

Let y = 1 so that y \in \mathbb{Z}.

For any x \in \mathbb{Z}, f(x) = 2x is always even, so there is no x such that f(x) = 1 = y, since 1 is odd.

Thus, there exists y \in \mathbb{Z} such that for all x \in \mathbb{Z}, f(x) \neq y.

Therefore, f is not surjective. QED.

\bigskip

Note: You may realize that f(A) = \{ f(x) \mid x \in A \} is just the image (range) of the function by definition, so another way to define surjectivity is to say f(A) = B. Since B \subseteq f(A) and f(A) \subseteq B, but because the image is always a subset of the codomain of a function, f(A) \subseteq B is trivial, so we really only need to prove B \subseteq f(A).

\bigskip

This is just a general approach, and not every question will be solvable by this exact approach. A single algorithm does not exist that works for all surjectivity proofs. Sometimes you will have to use your head more and work with the existing definitions you have.

\bigskip
